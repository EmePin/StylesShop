## Getting Started

Para obtener una copia local de este repositorio se puede clonar o [descargar](https://github.com/EmePin/StylesShop/archive/refs/heads/main.zip).
### Prerequisites

```sh
  git clone https://github.com/EmePin/StylesShop.git
```

-Abrir el archivo `index.html` en [vsc](https://code.visualstudio.com/) con la extensión [LiveServer](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)

### Installation


1.  Crear una cuenta de negocios en [paypal](https://paypal.com)
2.  Entrar a [paypal developer dashboard](https://developer.paypal.com/dashboard/) y crear una app para que se proporcione un cliente id y dos cuentas de prueba en el modo sandbox.
3.  Reemplazar en el head de carrito.html el client-id

      ```sh
            <script
                src="https://www.paypal.com/sdk jsclient-id=CHANGE_YOUR_ID_CLIENT&currency=MXN">
            </script>
      ```
